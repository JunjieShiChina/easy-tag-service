# easy-tag-service
easy-tag服务整合

easy-tag-service是基于easy-tag-core的基础上实现标签的配置化，存储化。可以将之作为配置标签的中台服务，对标签进行管理和功能增强
---
# 什么是easy-tag和easy-tag-core
见 [easy-tag-core](https://github.com/JunjieShiChina/easy-tag-core "easy-tag-core")
---
# easy-tag-service可以做什么
1.给规则定义标签

2.用标签匹配数据

3.设置标签模板

4.设置标签数据来源

5.标签功能增强
